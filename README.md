# msontran.github.io

Personal website for Son Manh Tran, served at **https://msontran.github.io**.

Static single-page site — `index.html` plus the CV PDF, no build step.

## Editing

Edit `index.html` and push to `main`. GitHub Pages redeploys automatically
(usually within a minute). To update the CV, replace `Son-Manh-Tran.pdf`.

## Local preview

```sh
python3 -m http.server
# then visit http://localhost:8000
```
