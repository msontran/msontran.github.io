# msontran.github.io

Personal website for Son Manh Tran, served at **https://msontran.github.io**.

Built with [Jekyll](https://jekyllrb.com/) and the
[Minima](https://github.com/jekyll/minima) theme, hosted on GitHub Pages.

## Structure

- `index.md` — home page content
- `_config.yml` — site + theme settings
- `Son-Manh-Tran.pdf` — CV (linked from the home page)

## Editing

Edit `index.md` and push to `main`. GitHub Pages rebuilds and redeploys
automatically (usually within a minute or two).

## Local preview

```sh
bundle install
bundle exec jekyll serve
# then visit http://localhost:4000
```
